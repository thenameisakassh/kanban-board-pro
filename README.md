📌 Kanban Board Pro
A modern drag-and-drop Kanban task management board built using HTML, CSS, and JavaScript.
This project allows users to organize tasks visually across multiple stages with a smooth UI and persistent storage.

🚀 Features
📋 Create tasks directly inside columns
✏️ Edit existing tasks
🗑 Delete tasks
🎯 Priority labels (Low / Medium / High)
📅 Due date tracking
🖱 Drag and drop tasks between columns
💾 LocalStorage support (tasks persist after refresh)
🌙 Dark themed modern UI
✨ Smooth animations and hover effects
📱 Responsive board layout

🛠 Tech Stack
HTML5 – structure
CSS3 – styling and animations
JavaScript (ES6) – application logic
LocalStorage API – persistent data storage

🎮 How It Works
Add a task inside any column.
Assign priority and due date.
Drag tasks between Todo, In Progress, and Done columns.
Edit or delete tasks anytime.
Tasks are automatically saved using LocalStorage.
